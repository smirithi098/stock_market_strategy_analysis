# Analysis of various trading strategies on Nifty Bank stocks

The idea behind the project is to apply some of the popular strategies (using technical indicators) to the Indian stock market to assess the profitability of these strategies and the returns generated at the end of the investment period. Additionally, through decision tree classification model, the aim is to predict trading signals in real-time by training the model with past data. SMOTE analysis is used as an improvement to the base model to balance the class lables and enhance the efficiency of the base model.

## Stocks picked:
- Axis Bank (Banking)
- ITC (Commodities)
- Reliance Indiustries (Gas & Petroleum)
- TCS (Technology)
- Airtel (Telecommunication)

## Technical indicators used:
- EMA
- RSI
- MACD
- Bollinger bands
